 Interactive Business Dashboard

This project is an interactive dashboard that visualizes key business metrics like sales, profit, and performance trends. It helps users explore data easily and make better decisions.

 Objective

To transform raw business data into meaningful insights using interactive visualizations.

Tools Used

Python, Pandas, Matplotlib, Seaborn, Plotly

 Outcome

Provides a clear view of business performance and supports data-driven decision-making using techniques from Data Science.
